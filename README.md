## About The Project

"失智列車" 泛指廣大網路鄉民在情報尚未明朗、調查不夠充分時，
因為新聞媒體的腥羶色報導而妄下評斷、嘲諷叫罵，最終案情卻大翻盤的狀況。

因大眾媒體識讀觀念尚未完全普級，
本站以巴哈姆特-場外休憩區為例，彙整討論板的各項失智列車事件。
希望大眾引以為戒，以審慎且不放棄思考的心態看待台灣的新聞。

請注意，雖然視聽者輕率進行聲討固然錯誤，
但新聞台之報導為國家資訊推送主力，對於這般亂象需負起較大責任，
政府機關也應善盡監督新聞內容之職責，而非將問題全部歸咎於網路論壇及人民。
